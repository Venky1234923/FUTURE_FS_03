
# 🍛 Spice Heritage Restaurant Website

A modern and responsive restaurant website built using **HTML, CSS, and JavaScript**. This project showcases an authentic Indian restaurant with an attractive user interface, menu display, gallery, reservation form, customer reviews, and contact information.

## 📌 Features

- Responsive Navigation Bar
- Attractive Hero Section
- About Us Section
- Today's Special Dish
- Searchable Food Menu
- Family Combo Offers
- Catering Services Section
- Customer Statistics Counter
- Food Gallery
- Customer Reviews
- Table Reservation Form
- Contact Information with Google Maps
- WhatsApp Contact Button
- Scroll-to-Top Button
- Smooth Navigation Effects
- Mobile Responsive Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome Icons




### 🏠 Home
Welcome section with restaurant introduction and call-to-action buttons.

### ℹ️ About Us
Information about the restaurant's heritage, values, and food quality.

### 🍽️ Menu
Displays food items with prices and includes a search feature.

### 🎁 Family Combo Offers
Special meal packages for families and groups.

### 🎉 Catering Services
Services for:
- Birthday Parties
- Wedding Events
- Corporate Events
- House Warming Functions

### 📷 Gallery
Food gallery showcasing delicious dishes.

### ⭐ Reviews
Customer feedback and ratings.

### 📅 Reservation
Online table reservation form.

### 📞 Contact
Restaurant location, phone number, email, opening hours, and Google Maps integration.

## ⚡ JavaScript Functionalities

- Welcome Popup Message
- Reservation Form Submission Alert
- Menu Search Filter
- Scroll-to-Top Button
- Active Navigation Highlighting
